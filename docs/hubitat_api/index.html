Common Endpoints

EndpointDescription
/devicesList authorized devices (id, name, label)
/devices/allFull device details (capabilities, attributes, commands)
/devices/{id}Single device details
/devices/{id}/{command}Execute device command
/devices/{id}/{command}/{value}Command with parameter(s)
/devices/{id}/eventsRecent device events
/devices/{id}/attribute/{name}Single attribute value
/modesList/get active mode
/modes/{id}Set mode
/hsmGet HSM status
/hsm/{command}Arm/disarm HSM
/hubvariablesList hub variables
/postURL/{url}Set webhook URL
